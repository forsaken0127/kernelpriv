# kernelpriv - 内核安全扫描程序

## Disclaimer

**kernelpriv** 是一个 Python 脚本，仅用于教育和道德目的。该工具扫描最新的内核版本，并在本地系统上测试各种已知的根漏洞。它只能在明确许可的情况下使用，并用于合法的、授权的渗透测试。

**注意：** 未经授权访问计算机系统、网络或数据是非法和不道德的。**kernelpriv** 的作者对该工具造成的任何误用或损坏概不负责。

## Features

- 快速扫描可用的最新内核版本。 - 出于教育和道德目的，针对本地系统测试各种已知的根漏洞。
